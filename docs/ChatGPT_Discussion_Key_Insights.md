# 📚 ChatGPT 대화 핵심 인사이트 정리

## 1. 🤖 AI 모델 및 인프라

### 1.1 모델 비교
| 모델 | 스펙 | 실행 환경 | 추론 능력 | 비용 |
|------|------|-----------|-----------|------|
| **gpt-oss-20B** | 200억 파라미터, Dense | M4 Max 가능 (양자화) | GPT-3.5 하위 | 저렴 |
| **gpt-oss-120B** | 1170억 파라미터, MoE | 80GB GPU 필수 | GPT-3.5+ | 중간 |
| **Qwen3-Coder-480B** | 4800억 파라미터, MoE | 클러스터급 | GPT-4 초반 | 높음 |

### 1.2 핵심 인사이트
- **맥북 M4 Max 한계**: 7B~20B 모델까지만 로컬 실행 가능
- **120B 모델**: 프리트레인 상태로 GPT-3.5급, 파인튜닝 필요
- **클라우드 비용**: A100 시간당 $0.78~$1.74, H100 $2.69~$2.74

### 1.3 파인튜닝 전략
```python
# LoRA/QLoRA 방식이 현실적
- Full Fine-tuning: 불가능 (수천 GPU 필요)
- LoRA: A100 80GB 1~2장으로 가능
- 비용: 50~100시간 학습 시 $80~$150
```
