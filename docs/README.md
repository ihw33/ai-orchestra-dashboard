## 🚀 Quick Start

### 1. 프로젝트 이해하기
1. [PROJECT_STATUS.md](PROJECT_STATUS.md)로 현재 진행 상황 파악
2. [AI_TEAM_MANUAL.md](AI_TEAM_MANUAL.md)로 팀 구성 이해
3. [TECHNICAL_ARCHITECTURE.md](TECHNICAL_ARCHITECTURE.md)로 시스템 구조 학습

### 2. AI 팀원 역할
- **Gemini**: 데이터 분석 & 시스템 모니터링
- **Codex**: API 개발 & 자동화
- **Cursor**: UI/UX & 프론트엔드
- **Claude**: 코드 리뷰 & 품질 보증
- **Task Master**: 문서화 & 작업 관리

### 3. 주요 기능
- GitHub Issues 자동 관리
- 실시간 작업 진행 추적
- AI 팀원 간 자동 통신
- WebSocket 실시간 업데이트
